TO compile the Maven project use the following command in the project4 directory:

$ mvn compile

To test the project with test classes in the test subdirectory,
run the following command in the project4 directory:

$ mvn test

To package your project, once the project has been tested and works properly,
run the following command in the project4 directory:

$ mvn package

To generate the online documentation (a website) of the project, having properly formatted
Javadoc comments, run the following command in the project4 directory:

$ mvn site

TO clean the project (remove the complied .class files, the jar file, and other 
created artifacts) run the following command in the project4 directory:

$ mvn clean
